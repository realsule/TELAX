# 🌾 TELAX Agricultural Marketplace

## 📖 Description
TELAX is a modern agricultural marketplace that connects farmers directly with buyers using cutting-edge technology. Built with React and Python, featuring a beautiful glassmorphism UI with agricultural-themed design.

## 🚀 Features

### 🌱 Core Functionality
- **User Authentication**: Secure login/registration system
- **Multi-Role Dashboard**: Farmer, Buyer, and Admin interfaces
- **Product Listings**: Browse and manage agricultural products
- **Payment on Delivery (POD)**: Secure payment system
- **Real-time Orders**: Track orders from farm to table

### 🎨 UI/UX Features
- **Glassmorphism Design**: Modern, elegant interface
- **TLX Agricultural Theme**: Forest Green, Soil Terracotta palette
- **Dark/Light Mode**: Seamless theme switching
- **Responsive Design**: Works on all devices
- **Smooth Animations**: Professional micro-interactions

### 🛠 Technical Features
- **Error Handling**: Defensive programming prevents crashes
- **Component Reusability**: Modular, maintainable code
- **State Management**: Redux for efficient data flow
- **API Integration**: RESTful backend architecture

## 🏗 Tech Stack

### Frontend
- **React 19.x** - Modern UI framework
- **Redux Toolkit** - State management
- **React Router v7** - Navigation
- **Tailwind CSS v3.4.1** - Styling
- **Vite 7.x** - Build tool
- **Lucide React** - Icons

### Backend
- **Python** - Backend language
- **Flask/Django** - Web framework
- **PostgreSQL** - Database
- **JWT** - Authentication

## 📦 Installation

### Prerequisites
- Node.js 18+
- Python 3.9+
- PostgreSQL

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app/main.py
```

## 🔧 Environment Variables

Create `.env` file in root directory:
```
DATABASE_URL=postgresql://username:password@localhost/telax
JWT_SECRET=your-secret-key
API_URL=http://localhost:8000
```
